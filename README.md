## Usage

To run the example project, clone the repo, and run `pod install` from the Example directory first.

## Requirements

- iOS 16.1+
- Xcode 14.1

## Features

- [x] Currency Conversion
- [x] Commission Fee
- [x] Validation

## Technical Features

- [x] Swift/SwiftUI
- [x] CoreData
- [x] API Manager
- [x] Unit Tests
